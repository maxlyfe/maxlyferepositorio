Blue Soul

BlueSoul é um game de ficção interativa feito para o projeto final do módulo de lógica de programação.

No qual você é um plebeu que deseja se tornar um grande cavalheiro, sua jornada começa quando se depara frente a morte e as suas decições vão fazer a diferença para conquistar o seu sonho ou continuar com a mesma vida que você leva até hoje.

O jogo consiste em escolher uma dentro de tres opções, pedra, papel ou tesoura.

Simulador de Votos

Projeto modulo 01 Blue EdTech

Em este projeto realizamos um programa em Python, onde serão informados X quantidade de candidatos e será efetuado um sistema de votos até ser informado que não tem mais ninguem para votar. Uma vez que finalizar a votação o sistema conta quem teve mais votos, informa o ganhador e informa quantos votos cada candidato obteve.
